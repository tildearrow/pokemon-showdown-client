Asuka sound engine!

hereby I would like to announce that this instance has now been ported to use the new in-house Asuka sound engine!
this should completely eliminate gaps in music playback (that even Gapless-5 couldn't deal with), fix sound issues on Safari (well, kind of), and act as a proof-of-concept for the following:

The main complicated thing we do with sound is loop music with an intro.
This is unfortunately not supported by ANY sound library out there
(I had to manually add support for it myself to soundManager!)

In the end, I don't think the existing libraries out there actually give us anything I care about.
